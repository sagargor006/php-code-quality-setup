# PHP Code Quality Setup

Two scripts to set up PHP code quality tooling — one for your IDE, one per project.

---

## Scripts

| Script | Purpose | Run |
|--------|---------|-----|
| `setup-ide.sh` | Install IDE extensions and settings | Once per machine |
| `setup-project.sh` | Copy config files and install Composer packages | Once per project |

---

## Docs

- [IDE Setup](./IDE-SETUP.md) — extensions, settings, supported IDEs (VSCode / Antigravity / Cursor)
- [Project Setup](./PROJECT-SETUP.md) — config files, Composer packages, pre-commit hooks (Laravel / CodeIgniter 3)
